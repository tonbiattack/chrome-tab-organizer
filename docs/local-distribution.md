# Local Usage Guide

このドキュメントは、Chrome Web Store を使わずに `git clone` または ZIP 配布でこの拡張を利用する場合の説明をまとめたものです。

## 結論

ローカル利用だけであれば、Chrome Web Store 向けの掲載素材や審査向け資料は必須ではありません。

不要なもの:

- ストア掲載文
- スクリーンショット
- Chrome Web Store の Privacy タブ用回答
- 審査担当向けの補足説明

あった方がよいもの:

- セットアップ手順
- 必要権限の説明
- 保存する設定の説明
- 制約事項

## この拡張が扱う情報

この拡張は、開いているタブの URL をローカルで読み取り、次の用途に使います。

- 重複 URL の判定
- 標準ルールによるサービス判定
- カスタムルールによる Jira 課題キー判定

外部送信は行いません。

## 保存するもの

`chrome.storage.local` に次を保存します。

- `全ウィンドウを対象にする` の状態
- 標準ルールの有効 / 無効
- カスタムルール

カスタムルールには次が含まれます。

- グループ名
- 色
- ルール種別
- Jira 課題キーまたは URL パターン
- 有効 / 無効

## 利用者向けに最低限伝えるとよい内容

1. この拡張はタブ URL をローカルで見て整理する
2. 設定とカスタムルールはブラウザ内に保存する
3. 外部サーバーには送信しない
4. 既存のタブグループは整理時に解除して再作成する
5. Jira の親子関係は自動取得ではなく、現時点では課題キー集合を手動定義する

## チーム利用時の補足

チームや組織内で配布する場合は、README に加えてこのドキュメントを共有しておくと、権限や保存内容を説明しやすくなります。  
公開ストア向けの formal な privacy policy までは通常不要ですが、利用者に対して次を明確にすることが推奨されます。

- この拡張が何を読み取るか
- 何を保存するか
- 外部送信の有無
